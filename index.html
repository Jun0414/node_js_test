    npm i(nstall) : package.json 안에 있는 devDependencies 와 dependencies의 모듈을 사용하는데
                    필요한 모듈을 찾아서 설치해준다.
    
    npm run [script] : package.json 안에서 script에 작성한 것을 실행하며 보통 devDependencies에 설치한 모듈을
                      실행하는 코드를 만들고자 생성 후 실행한다. (이 또한 필요한 것을 자동으로 생성해준다.)

    .gitignore : 버전화 시킬 필요가 없는 모듈이나 패키지를 제외하고 싶을때 원하는 것을 목록에 작성해주면 된다.
